7 Engines & content JSON schema

The core architecture: each activity is a JSON object; one widget per type renders it. This is what Adey Studio (CMS) produces and the app consumes. Build the 6 engine widgets against these shapes.

Shared envelope (every activity)

{
  "id": "learn-001",
  "type": "tracing",            // matching|quiz|tracing|dragdrop|story|coloring
  "title": "Trace the Fidel",
  "category": "learn",          // play|learn|watch|create
  "character": "adey",
  "age_min": 3, "age_max": 6,
  "language": "en",             // en|am|both  (am deferred)
  "is_free": true,
  "status": "live",             // live|draft|scheduled|archived
  "media": { "thumbnail": "...", "background": "...", "narration": "..." },
  "reward_stars": 3,
  "content": { /* engine-specific, below */ }
}

matching

"content": {
  "grid": "4x3",
  "pairs": [
    {"label":"Lion","image":"lion.png",
     "audio":"lion.mp3"}, ...
  ]   // 4–16 pairs
}

quiz

"content": {
  "questions": [
    {"prompt":"Which says moo?",
     "choices":["Cow","Dog","Cat","Lion"],
     "correct":0,"reaction":"celebrate"}
  ]   // min 3
}

tracing

"content": {
  "glyphs":[
    {"char":"ሀ","roman":"ha",
     "stroke_order":"1→2→3",
     "audio":"ha.mp3"}
  ]
}

"content": {
  "objects":[
    {"label":"Teff","image":"teff.png",
     "target":"pot"}
  ],
  "zones":["pot"]
}

story

"content": {
  "scenes":[
    {"bg":"savanna.png","text":"...",
     "narration":"s1.mp3","duration":6,
     "choices":[{"label":"Mountain",
       "goto":3}]}
  ]
}

coloring

"content": {
  "svg":"lion-outline.svg",
  "regions":["mane","face","ear"],
  "palette":["#E23B3B","#F2A23B",
    "#FFD23B","#34C06B", ...]
  // 6–16 colors

Why this matters: new content ships as JSON + assets — no app release. The CMS (Adey Studio) is already designed around exactly these shapes, so the contract is fixed. Lock this schema first; everything else depends on it.

9 Functional rules

Onboarding & auth

  • Parent account: email or Google (Supabase Auth)
  • Child profile: name, age 2–10, avatar (1 of 4)
  • Up to 3 child profiles; isolated progress
  • No child PII beyond name + age

Parent gate

  • Math challenge (a + b) before parent area & any purchase
  • Wrong answer → shake, retry; no lockout
  • Required by PRD before settings/purchase

Paywall & subscription

  • 7-day free trial
  • Yearly $39.99 / Monthly $4.99
  • 8 free activities; rest premium (lock badge)
  • Restore purchases; store product IDs TBD
  • Suggested: RevenueCat

Progress & rewards

  • Stars per activity (1–3 by performance)
  • Daily streak; daily surprise box
  • Completion criteria per engine (e.g. all glyphs traced / all pairs found)
  • Track per child; surface in Parent area

Screen-time & safety

  • Daily limit: 20/30/45/60 min or off
  • No ads, ever; no external links in kid UI
  • COPPA / KidSAFE compliance review required

Audio behavior

  • Narration auto-plays on screen/activity load
  • Tap = SFX; correct/wrong/celebrate cues
  • Sound & music independently toggleable (Settings)
  • Respect device silent / mute
